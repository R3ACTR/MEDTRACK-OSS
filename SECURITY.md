# MedTrack OSS Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open a public issue. Instead, email us at **security@medtrack-oss.org**.

We aim to respond within 48 hours and provide a fix within 7 days.
