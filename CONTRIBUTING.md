# Contributing to MedTrack OSS

First off, thank you for considering contributing to MedTrack OSS! It's people like you that make MedTrack OSS such a great tool.

### Reporting Bugs
- Use the GitHub Issues tab.
- Describe the bug and steps to reproduce.
- Include OS version and browser details.

### Suggesting Enhancements
- Open a feature request in Issues.
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo.
2. Create a new branch: `git checkout -b feature/your-feature-name`.
3. Commit your changes: `git commit -m 'Add some feature'`.
4. Push to the branch: `git push origin feature/your-feature-name`.
5. Open a Pull Request.
